# 🛡️ Security Controls

## 🧠 What are Security Controls?

Security controls are safeguards or mechanisms used to:

- Prevent attacks  
- Detect threats  
- Reduce damage  
- Recover from incidents  

They are applied to protect:

- Digital systems  
- Physical infrastructure  
- Human interactions  

---

## 🎯 Why Controls Matter

The CIA Triad defines *what we want to protect*:

- Confidentiality  
- Integrity  
- Availability  

👉 Controls define *HOW we protect them*

---

## 🔒 Controls for Confidentiality

### 🎯 Goal:
Ensure that only authorized users can access sensitive data.

---

### 🛡️ Common Controls:

#### 🔐 Encryption
Transforms readable data into unreadable format.

👉 Example:
Messages sent through secure apps are encrypted so outsiders cannot read them.

---

#### 👤 Access Control
Restricts who can view or use data.

👉 Example:
Only instructors can access exam question banks in a university system.

---

#### 🔄 Patch Management
Regularly updating software to fix vulnerabilities.

👉 Example:
Updating an operating system to remove known security weaknesses.

---

## 🧾 Controls for Integrity

### 🎯 Goal:
Ensure data remains accurate and unchanged unless authorized.

---

### 🛡️ Common Controls:

#### 🔢 Checksums / Hashing
Used to detect if data has been altered.

👉 Example:
Downloaded files often include a hash value to verify correctness.

---

#### 🔑 Permission Control
Limits who can modify data.

👉 Example:
Only database admins can edit critical system records.

---

#### 💾 Data Backup
Stores copies of data for recovery.

👉 Example:
Restoring a database after accidental deletion.

---

#### 📜 Audit Trails
Logs all changes made to data.

👉 Example:
Tracking who updated a file and when the change occurred.

---

## ⏱️ Controls for Availability

### 🎯 Goal:
Ensure systems and data are accessible when needed.

---

### 🛡️ Common Controls:

#### 🖥️ Redundancy
Using multiple systems to avoid single points of failure.

👉 Example:
A website hosted on multiple servers to handle traffic or failure.

---

#### 🛡️ Protective Software
Prevents disruptions caused by attacks.

👉 Example:
Firewall blocking malicious traffic.

---

#### 🔁 Disaster Recovery Plans
Prepared procedures to restore systems quickly.

👉 Example:
Recovering services after a power outage or system crash.

---

## ⚖️ Important Insight

Some controls support multiple objectives.

👉 Example:
Access control helps both:
- Confidentiality (who can see data)
- Integrity (who can modify data)

---

## 🎯 Key Takeaway

Security controls are the practical tools and strategies used to:

- Protect data  
- Maintain accuracy  
- Ensure system availability  

---

## 🧠 Key Insight

In real systems, the effectiveness of cybersecurity depends not only on having controls, but on how well they are combined and maintained over time.

For example, strong encryption is useless if access credentials are weak or shared improperly.
